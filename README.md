# Gunevo Tower Battle

ガンエボのファンゲーム
ガンエボ版どうぶつタワーバトル

## 技術スタック

- Phaser 3.88.2 + Matter.js
- React 19 + Vite
- Trystero（P2P 通信）
- TypeScript

## ゲームリンク
gunevo-tower-battle.devkey.jp

## 基本操作

- マウス移動: ピースの左右移動
- 左クリック: ドロップ
- 右クリック: 回転

## オンライン対戦

1. Create Room で部屋を作成
2. 生成された ID を相手に共有
3. Join Room で参加
